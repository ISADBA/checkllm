# 快速开始

## 构建

```bash
env GOCACHE=/tmp/go-cache-checkllm-build go build ./...
```

## 运行

```bash
go run ./cmd/checkllm run \
  --base-url https://api.openai.com/v1 \
  --api-key $OPENAI_API_KEY \
  --model gpt-5.4 \
  --baseline docs/baselines/openai-gpt-5.4.md \
  --output docs/runs/local-gpt-5.4.md
```

## 输出

执行完成后会：

- 在终端输出核心分数与结论
- 在 `docs/runs/*.md` 生成完整鉴定档案，包含 probe 输入、期望、原始输出、tool calls、stream events 等
- 在 `docs/repos/*.md` 生成面向用户阅读的解释型报告

## 当前限制

- 仅支持 OpenAI 兼容 `/responses` 接口
- 仅支持文本类 probe
- `usage` 校验当前基于返回字段和本地粗粒度估算
- 流式解析当前只实现第一版 SSE 事件抽取
